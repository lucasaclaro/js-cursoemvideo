Formatando strings

.length - quantidade de caracteres da string

.toUpperCase() - tudo para letras maiúsculas

.toLowerCase() - tudo para letras minúsculas

Formatando números

.toFixed(2) - quantidade de casas decimais a ser informada dentro dos parênteses

.replace(' . ', ' , ') - é possível substituir uma coisa por outra, por exemplo, um ponto por uma vírgula
